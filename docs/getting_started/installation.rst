Installation
============

.. toctree::
   :maxdepth: 1

   
NetOpsAuto suppoorts Python 3.85+. The reccomended way to clone the `git repository<https://github.com/dmtx97/netopsauto.git>`_ and run the setup.py in a new virtual environment.


Create new virtual environment within the netopsauto repository:

.. code-block:: RST
   python -m venv env

Activate the virtual environment:

.. code-block:: RST
   python -m venv env
